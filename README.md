<!-- GETTING STARTED -->
### Prerequisites

 Make sure python3 is installed. To check run the command :
* python
  ```sh
  python --version
  ``` 
  OR 
  ```sh
  python3 --version
  ```
  
## Getting Started

To get a local copy up and running follow these steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CYCLOP5/Calculator-Team2.git
   ```
2. Get into the directory
    ```sh
    cd Calculator-Team2
   ```
